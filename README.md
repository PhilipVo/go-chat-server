# Go Chat Server

Server written in Golang with websocket implementation to interface with React frontend.

## Getting Started

### Installing

Install the additional packages

```
go get github.com/stretchr/testify/assert
go get github.com/gorilla/websocket
```

Compile and run

```
go run main.go
```
